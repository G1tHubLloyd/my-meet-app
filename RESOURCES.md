## üöÄ Deployment Platforms

### Free Hosting Options
1. **GitHub Pages** (What we're using)
   - Free tier: 1 site per account
   - Static hosting
   - Custom domain support

2. **Vercel**
   - Free tier: Generous limits
   - Serverless functions included
   - Better performance

3. **Netlify**
   - Free tier: Similar to Vercel
   - Build process automation
   - Form handling

### Serverless Platforms
1. **AWS Lambda** (Recommended)
   - Pay-per-use pricing
   - 1M free requests/month
   - Best for production

2. **Firebase Cloud Functions**
   - Alternative to Lambda
   - Good Google integration

3. **Azure Functions**
   - Enterprise option
   - Microsoft ecosystem

## üìö Cheat Sheets

### Git Commands
```bash
git init                    # Initialize repository
git add .                   # Stage all changes
git commit -m "message"     # Create commit
git push origin main        # Push to GitHub
git pull origin main        # Pull from GitHub
git log --oneline          # View history
```

### npm Commands
```bash
npm init                    # Create package.json
npm install package         # Install package
npm install --save-dev pkg  # Install dev dependency
npm run script             # Run script from package.json
npm update                 # Update all packages
npm outdated               # Show outdated packages
```
